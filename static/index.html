일본경마 통합 분석 (배당판 + 전적표)

배당판은 Alt+C 화면 캡처 또는 파일, 전적표는 이미지 업로드. 하나를 바꿔도 다른 하나는 유지됩니다.

🎰 배당판⬜ 미설정
📋 전적표⬜ 미설정
배당판과 전적표를 모두 준비하세요

배당판 캡처 & 이상배당 감지

Alt+C로 현재 화면을 즉시 캡처할 수 있습니다(첫 1회 화면공유 허용 후 팝업 없이 재캡처). 또는 [화면 캡처]/이미지 파일 업로드 → 영역 드래그 → 분석.

배당판 영역을 마우스로 드래그해 선택하세요. 선택하지 않으면 전체를 분석합니다.

📡 배당 시계열 이상감지 (1차/2차 캡처)

화면을 캡처/크롭한 상태에서 ① 베팅종류 선택 → ② [1차 캡처]·[2차 캡처]로 두 시점의 배당을 저장 → ③ [이상감지 분석].

먼저 화면을 캡처/크롭한 뒤 [1차 캡처]를 누르세요.

🎯 통합 분석 (전적 + 배당 이상감지)

한국경마 PDF로 분석한 경주를 선택하고, (배당판 캡처 탭에서 같은 경주명으로 1·2차 배당을 저장했다면) 예산을 입력해 [통합 분석]을 누르세요. 전적 점수 + 배당 신호 → 등급·베팅을 한 화면에 표시합니다.

먼저 한국경마 탭에서 경주를 분석하세요.

경주 결과 입력

실제 착순을 입력하면 누적 학습 DB에 반영됩니다.
