경주 상세

📅 최근 결과 요약

📸 배당판 스냅샷 갤러리 마감 1분전 자동 + 수동 캡처

확장(오버레이) 📸 버튼 또는 마감 1분 전 자동 캡처로 저장됩니다. 썸네일을 누르면 크게 보이고, 실제 결과가 있으면 추천 조합과 1·2·3착을 비교합니다.

📋 결과 입력 대기

분석·배당 수집된 경주가 자동으로 여기 뜹니다. [결과 입력]을 누르면 팝업에서 착순·배당만 넣어 즉시 적중판정·학습됩니다.

✏️ 경주 결과 직접 입력

실제 착순을 입력하면 누적 학습 DB에 반영됩니다.

📝 수동 추천 저장 라이브 배당 없어도 전문가 추천을 기록

배당이 수집 안 된 경주에도 복승/삼복승 추천을 저장하면, 나중에 결과를 입력할 때 자동으로 적중 판정됩니다.

최근 결과를 날짜별로 모아 보여줍니다. 경주를 클릭하면 적중은 "왜 맞았는지", 미적중은 "왜 놓쳤는지" 복기가 열립니다.

📋 일괄 결과 등록

결과 페이지(asyukk34 등)의 URL을 넣거나 결과표 HTML을 붙여넣으면, 모든 경주의 1~3착·확정배당을 한 번에 파싱해 분석한 경주와 자동 매칭·적중판정·학습합니다.

URL은 Chrome 확장(로그인 세션)을 경유해 가져옵니다(확장 설치 + 배당판 로그인 필요). 실패하면 결과 페이지에서 전체 선택(Ctrl+A)→복사→아래에 붙여넣기 후 [붙여넣기 등록].

📋 엑셀/스프레드시트 표를 그대로 복사해 붙여넣어도 됩니다(탭 구분 텍스트). 첫 줄이 헤더(구분/경주지역/라운드/1착/2착/3착/단승/복승/…/삼복승)면 자동 인식 → 경마장+경주번호로 분석 로그와 매칭.

📸 결과 사진 첨부 → 자동 반영

경주결과 화면 사진(스크린샷)을 첨부하면 표의 모든 경주(1~3착·복승/삼복승 배당)를 자동 판독해 분석한 경주와 매칭·적중판정·학습합니다. 확장·로그인 불필요.

⚡ 순서대로 빠른 입력

분석한 경주를 시간순으로 나열합니다. 경주명은 자동 매칭되므로 1~3착(+4착·배당)만 순서대로 입력하고 [저장]하면 즉시 적중판정·복기가 실행됩니다.
