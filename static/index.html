배당판 캡처 & 이상배당 감지

Alt+C로 현재 화면을 즉시 캡처할 수 있습니다(첫 1회 화면공유 허용 후 팝업 없이 재캡처). 또는 [화면 캡처]/이미지 파일 업로드 → 영역 드래그 → 분석.

배당판 영역을 마우스로 드래그해 선택하세요. 선택하지 않으면 전체를 분석합니다.

📡 배당 시계열 다중 캡처 · 실시간 분석 (복승+쌍승)

화면 캡처/크롭 후 [📸 캡처 누적] 반복 또는 [🔀 복승+쌍승 캡처]로 두 배당을 한 번에. 자동 캡처(2/3/5분)는 복승/쌍승을 교대로 누적합니다.

1 [복승] 탭 열기
→
2 [🔀 복승+쌍승 캡처] 누르기
→
3 안내 뜨면 [쌍승] 탭으로 전환 (자동 캡처)
자동 캡처 홀수=복승 · 짝수="쌍승 전환" 안내 후 5초 뒤 자동 캡처

⌨️ 빠른입력 — Vision 없이 즉시 계산(0.1초). 한 줄에 마번 배당 (예: 4 2.4). 콤마/줄바꿈 구분, 번/배 같은 글자는 무시됩니다.

입력할 때마다 시계열에 한 포인트씩 누적됩니다.

🔀 쌍승 빠른입력 — 한 줄에 A B A→B배당 B→A배당 (예: 3 7 4.2 6.1). 두 방향 배당의 대소가 뒤바뀌면 역전으로 표시됩니다.

🎰 복승 매트릭스 빠른입력 — 한 줄에 A B 배당 (예: 3 7 8.4). 순서무관 조합 배당을 삼각 매트릭스로 표시합니다.

[복승] 탭을 열고 [🔀 복승+쌍승 캡처]를 누르세요.

🎯 3종 동시 분석 (복승 · 쌍승 · 삼복승)

각 배당판을 띄우고 캡처하세요(현재 화면 전체). 3종을 함께 판독해 베팅종류 간 불일치(이상)를 감지합니다.

최소 1종 캡처 후 분석 가능
확장이 보낸 3종 배당을 매트릭스로 표시하고, 규칙기반 이상감지(급락·역전·유력마·삼복승추천)를 실행합니다.

🎯 통합 분석 (전적 + 배당 이상감지)

한국경마 PDF로 분석한 경주를 선택하고, (배당판 캡처 탭에서 같은 경주명으로 1·2차 배당을 저장했다면) 예산을 입력해 [통합 분석]을 누르세요. 전적 점수 + 배당 신호 → 등급·베팅을 한 화면에 표시합니다.

먼저 한국경마 탭에서 경주를 분석하세요.
